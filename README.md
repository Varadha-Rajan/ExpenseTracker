# 💰 Expense Tracker

A web-based Expense Tracker application developed using Java Full Stack technologies.  
This system helps users manage daily expenses, track spending patterns, and monitor financial activities efficiently.

---

## 🚀 Features

- 👤 User Registration & Login
- ➕ Add Daily Expenses
- 📝 Edit & Delete Expense Records
- 📅 Track Expenses by Date
- 📊 View Expense Summary
- 💵 Categorize Expenses (Food, Travel, Bills, etc.)
- 🔍 Search Expense History
- 🔒 Secure Authentication

---

## 🛠 Technologies Used

- ☕ Java (Servlet & JSP)
- 🗄 JDBC
- 🐬 MySQL Database
- 🌐 HTML, CSS, JavaScript
- 🖥 Apache Tomcat Server
- 🧰 Eclipse IDE

---

## 💻 System Requirements

- Java JDK 8 or higher
- Apache Tomcat 9+
- MySQL Server
- Eclipse IDE (or any Java IDE)
- Modern Web Browser

---

## ⚙️ How to Run the Project

1. Clone or download the repository
2. Import the project into Eclipse
3. Configure MySQL database
4. Update database username & password in code
5. Deploy the project on Apache Tomcat Server
6. Run the application in your browser
